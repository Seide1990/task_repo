Yerli repozitoriyanızı uzaq depo ilə əlaqələndirmək

Yerli repozitoriyanızı uzaq depo ilə əlaqələndirməküçün aşağıdakı ardıcıllığı izləyin

  1. Reponuzu yerləşdirmək istədiyiniz papkada yeni bir papka yaradın.
  2. yeni yaratdığınız papkanın içində olduğunuza əmin olun.
  3. " git init " əmri ilə qovluğu git qovluğuna çevirin.
  4. " touch index.html" əmri ilə papkanın içində index.html adlı fayl yaradın.
  5. " code ." əmi ilə index.html faylını Visual Studio Code İDE -sində açırıq.
  6. index.html faylında kodlarımızı daxil edirik
